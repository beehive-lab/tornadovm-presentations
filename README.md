# TornadoVM Presentations

This repository contains slides from talks, tutorials, workshops, conferences, and invited presentations related to [TornadoVM](https://www.tornadovm.org).

It is organized chronologically (by year, then venue, then event) so that presentations from the same venue across different years are kept cleanly separated.

## Repository layout

```text
slides/<year>/<venue>/<event-or-talk-title>/
```

Each event folder is self-contained and includes at minimum a `README.md` and `slides.pdf`. See [CONTRIBUTING.md](CONTRIBUTING.md) for the full conventions.

Machine-readable metadata lives in [`metadata/`](metadata/) and can be used to regenerate this index, build a website, or create venue/year indexes.
